# 🚀 YOLO Vision AI - Real-Time Object Detection System

![YOLO Vision AI](assets/images/banner.png)

A real-time **Computer Vision AI application** built using **YOLOv8, OpenCV, PyTorch, and Streamlit** for detecting objects from images, videos, and live webcam streams.

The system provides fast object detection with bounding boxes, confidence scores, and tracking capabilities through an interactive web interface.

# 🖥️ Application Modules

## 📷 Image Detection

Upload an image and detect multiple objects instantly using YOLOv8.

## 🎥 Video Detection

Process videos and generate detection results with bounding boxes.

## 📡 Live Webcam Detection

Perform real-time object detection using your camera.

## 🎯 Object Tracking

Track objects across video frames and maintain object identities.
