# 🤖 YOLO Vision AI

A professional Object Detection application built using **YOLOv8**, **OpenCV**, and **Streamlit**.

## 🚀 Features

- 📷 Image Detection
- 🎥 Video Detection
- 📹 Live Webcam Detection
- 🚗 Object Tracking
- 📊 Dashboard
- ⚙ Settings
- 📥 Download Detection Results

## 🛠 Tech Stack

- Python
- YOLOv8
- OpenCV
- Streamlit
- PyTorch
